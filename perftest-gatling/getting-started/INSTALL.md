Installation
============

Unzip all zip's flat into one folder. You should have a directory structure like this

```
/
+ git
+ ideaC-2018.1.win
+ IdeaICWorkdir
+ java
+ maven
+ visualvm_141
+ webshop
* webshop.bat
* webshop.sh
* workshop_idea.bat
```

Start IDE

    workshop_idea.bat
    
Start WebShop Launcher

    webshop.bat
    
Unzip Gatling into a folder & Start the recorder
